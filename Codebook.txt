Codebook for dfGalaxySAverages.txt database using run_Analysis.R
Lawrence Lau, October 24, 2014

The data contained in the dfGalaxySAverages dataset as produced by running the run_Analysis.R script are taken from observations and variables collected by Smartlab.  A full description of the original data can be obtained at:

http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones 

The raw data can be obtained at:

https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip
